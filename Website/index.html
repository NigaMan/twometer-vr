TwometerVR

Twometer is an open-source alternative to commercial VR tracking systems that aims to be low-cost, accessible and feature-rich.

Learn more
Features

Low cost Each controller costs less than $15

Compatible Play BeatSaber, VRChat and any other game that runs on SteamVR

Reuses equipment Works with many different readily available consumer webcams

Open-source Both hardware and software files can be found on GitHub

Releases
The latest version is 1.x, which can be found on GitHub. It works, but it may be glitchy and the VR experience may not be satisfying due to missing controls and/or drift.
Version 2.x is currently in the making which fixes all glitches, removes the drift and will give the controllers a complete design overhaul resulting in a much more enjoyable experience.
